# Notes App

A simple front-end **Notes App** where you can write notes, add them to the list, and delete them.

## Features

- Add notes using the **Add Me** button
- Keyboard support:
  - **Enter** (from textarea) to add a note

## Files

- `index.html` - App layout
- `style.css` - Styling (dark theme)
- `script.js` - Note add/delete logic + keyboard handling

## How to run

1. Open `index.html` in a browser.
2. Start typing in the textarea.
3. Click **Add Me** or use the keyboard shortcuts.

## Notes

- Notes are stored only in the DOM (not persisted). Refreshing the page clears the notes.
